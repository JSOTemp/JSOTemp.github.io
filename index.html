Overview

Julia Smooth Optimizers is an organization on GitHub containing a collection of Julia packages for Nonlinear Optimization software development, testing, and benchmarking. We provide tools for building models, access to repositories of problems, subproblem solving, linear algebra, and solving problems.

Below there is a brief summary of packages.

  • In blue, NLPModels.jl, which provides different ways to create nonlinear programming and nonlinear least squares models.
  • In green, packages providing access to problems from different collections.
  • In red, Optimize.jl containing implemented solvers. NLSCON is an unnanounced solver.
  • In yellow, linear algebra packages, including Krylov.jl with iterative solvers and many factorization packages.
  • In orange, LinearOperators.jl provides matrix-like entities and BenchmarkProfiles.jl provides performance and data profiles.
